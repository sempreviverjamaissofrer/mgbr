Política de Privacidade & Termos de Serviço para o bot MGBR no Discord.

Última atualização: — Plataforma: Discord Armazenamento: Banco de dados local (.db)

Política de Privacidade – MGBR

O bot MGBR, disponível na plataforma Discord, respeita a privacidade dos usuários e se compromete a proteger suas informações.

1. Coleta de Dados

O MGBR não coleta informações pessoais dos usuários além do estritamente necessário para o funcionamento correto de seus comandos e funcionalidades.

Os dados que possam ser processados (por exemplo: configurações de servidor, registros de moderação e preferências) são mantidos apenas para viabilizar o funcionamento do bot.

2. Uso de Dados

Qualquer informação processada pelo bot é utilizada exclusivamente para fornecer as funcionalidades solicitadas dentro do Discord e melhorar a estabilidade do serviço.

3. Armazenamento

Os dados ficam armazenados em um banco de dados local no formato .db. Não há transmissão de dados para fora do ambiente controlado pelo desenvolvedor.

4. Compartilhamento

O bot não compartilha, vende ou disponibiliza informações de usuários a terceiros.

5. Direitos dos Usuários

Você pode solicitar a remoção dos dados do seu servidor a qualquer momento. Basta entrar em contato pelos canais oficiais indicados abaixo.

6. Contato

Para dúvidas ou solicitações relacionadas à privacidade, utilize o Servidor oficial do MGBR no Discord.

Termos de Serviço – MGBR

Ao utilizar o bot MGBR no Discord, você concorda com os termos abaixo.

1. Uso do Bot

  • O bot está disponível gratuitamente na plataforma Discord.
  • O uso deve respeitar os Termos de Serviço do Discord e as Diretrizes da Comunidade.

2. Responsabilidades

  • O desenvolvedor do MGBR não se responsabiliza por danos, perdas ou problemas decorrentes do uso do bot.
  • O(a) administrador(a) do servidor é responsável pela forma como o bot é utilizado dentro de sua comunidade.

3. Armazenamento de Dados

  • O bot pode armazenar dados mínimos (como configurações e preferências) para garantir seu funcionamento.
  • Esses dados ficam restritos a um banco de dados local (.db) e não são utilizados para nenhum outro fim.

4. Modificações

  • O desenvolvedor pode atualizar o bot, esta Política de Privacidade e estes Termos a qualquer momento.
  • Mudanças relevantes serão comunicadas através do Servidor oficial do MGBR.

5. Contato

Para dúvidas, suporte ou solicitações, utilize o Servidor oficial do MGBR no Discord.
